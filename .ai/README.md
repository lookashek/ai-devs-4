# .ai — Agent Instructions

This directory stores instructions, prompts, and context files for AI agents working on this repository.

## Contents

| File | Purpose |
|---|---|
| `README.md` | This file — overview of the `.ai` directory |
| `lessons.md` | Instructions for implementing new lesson solutions |
| `conventions.md` | Detailed coding conventions and patterns |

## How to Use

When starting a new task with an AI agent, include the relevant file(s) from this directory in the agent's context alongside `CLAUDE.md` and `PROD.md`.

## Adding New Instructions

Create a new `.md` file in this directory when you need reusable instructions for a recurring task type. Keep files focused and task-specific.
